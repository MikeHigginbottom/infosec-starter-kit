# Internet Registry (IR)
- The IANA has the discretionary authority to delegate portions of its responsibility and, with respect to network address and Autonomous System identifiers, has lodged this responsibility with an IR. The IR function is performed by the DDN NIC. See also: Autonomous System, network address, Defense Data Network..., Internet Assigned Numbers Authority. [^1] #glossary
