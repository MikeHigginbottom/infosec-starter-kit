# Splunk
- A [SIEM](siem.md) tool
- Available from [Splunk](http://splunk.com/)
- Free version available but a bit restricted
- A very popular SIEM tool that enterprise companies use to correlate the data or events in their network. Although Splunk has a free tier, it has a limit to only ingest 500MB of data per day so prefer [Security Onion](security-onion.md) might be a [better option](siem-tool-comparison.md).
- If you want to expand that limit, you can request a developer license and it will be able to ingest 50GB of data per day. The license will need to be renewed every six months. Here is the link to request a license: [https://www.splunk.com/en_us/resources/personalized-dev-test-licenses.html](https://www.splunk.com/en_us/resources/personalized-dev-test-licenses.html)
- Splunk can ingest system data and logs and allow you to graph them. Deployment is simple, but configuring monitoring can be complex, but incredibly flexible.
