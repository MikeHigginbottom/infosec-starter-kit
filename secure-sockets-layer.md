# Secure Sockets Layer
- An encrypted method of communication between two endpoints, particularly Web sites. All versions of SSL are now considered insecure, and best practice is to use Transport Layer Security instead. [^1] The term [SSL](ssl.md) is however commonly used to refer to both SSL and TLS collectively. [^2] #glossary
