# gated
- Gatedaemon. A program which supports multiple routing protocols and protocol families. It may be used for routing, and makes an effective platform for routing protocol research. The software is freely available by anonymous FTP from `gated.cornell.edu`. Pronounced gate-dee. See also: Exterior Gateway Protocol, Open Shortest-Path First, Routing Information Protocol, routed.
