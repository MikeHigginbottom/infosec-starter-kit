# Time To Live
- A value in an Internet Protocol packet that tells a network router whether or not the packet has been in the network too long and should be discarded. [^1] #glossary 
- A field in the IP header which indicates how long this packet should be allowed to survive before being discarded. It is primarily used as a hop count. See also: Internet Protocol. [^2] #glossary
