# Cyber Chef
- A Swiss Army Knife for encryption, encoding, compression and data analysis
- Great for one-shot manipulations
- Can chain recipes together
- For reusability consider using [Chepy](chepy.md) which allows you to put the recipes in a Python script (you can save recipes for later though using the save button in the recipe panel)
- [GitHub - gchq/CyberChef: The Cyber Swiss Army Knife - a web app for encryption, encoding, compression and data analysis](https://github.com/gchq/cyberchef)
- Online version at https://gchq.github.io/CyberChef/
