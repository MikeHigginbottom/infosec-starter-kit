# Usenet
- A collection of thousands of topically named newsgroups, the computers which run the protocols, and the people who read and submit Usenet news. Not all Internet hosts subscribe to Usenet and not all Usenet hosts are on the Internet. See also: Network News Transfer Protocol, UNIX-to-UNIX CoPy. [^1] #glossary
