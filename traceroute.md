# Traceroute
- Traceroute is a tool the maps the route a packet takes from the local machine to a remote destination. [^1] #glossary 
- A program available on many systems which traces the path a packet takes to a destination. It is mostly used to debug routing problems between hosts. There is also a traceroute protocol defined in RFC 1393. [^2] #glossary
