# Archie
- A system to automatically gather, index and serve information on the Internet. The initial implementation of archie provided an indexed directory of filenames from all anonymous FTP archives on the Internet. Later versions provide other collections of information. See also: archive site, Gopher, Prospero, Wide Area Information Servers. [^1] #glossary
