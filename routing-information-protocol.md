# Routing Information Protocol
- Routing Information Protocol is a distance vector protocol used for interior gateway routing which uses hop count as the sole metric of a path's cost. [^1] #glossary 
- A distance vector, as opposed to link state, routing protocol. It is an Internet standard IGP defined in RFC 1058. See also: Interior Gateway Protocol, Open Shortest-Path First. [^2] #glossary
