# Random Number Generator
- A system that uses a source of genuinely random data in order to generate random numbers for consumption by computers. As sources of random data can be expensive to implement, [Pseudorandom Number Generators](pseudo-random-number-generator.md) are often used in their stead. [^1] #glossary
