# Gateway
- A network point that acts as an entrance to another network. [^1] #glossary 
- The term [router](router.md) is now used in place of the original definition of gateway. [^2]
- Currently, a gateway is a communications device/program which passes data between networks having similar functions but dissimilar implementations. This should not be confused with a protocol converter. By this definition, a router is a layer 3 (network layer) gateway, and a mail gateway is a layer 7 (application layer) gateway. [^2]
