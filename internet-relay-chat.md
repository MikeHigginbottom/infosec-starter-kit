# Internet Relay Chat
- An application layer protocol used for text communications in a client/server network. Created in 1988, IRC (IETF RFC 1459) is typically unsecured and has several well-documented vulnerabilities. [^1] #glossary
- A world-wide party line protocol that allows one to converse with others in real time. IRC is structured as a network of servers, each of which accepts connections from client programs, one per user. [^2] #glossary
