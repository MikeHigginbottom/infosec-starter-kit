# Network File System (NFS)
- A protocol developed by Sun Microsystems, and defined in RFC 1094 (RFC 1813 defines Version 3), which allows a computer system to access files over a network as if they were on its local disks. This protocol has been incorporated in products by more than two hundred companies, and is now a de facto Internet standard. [^1] #glossary
