# HTML
- Hypertext Markup Language is the standard markup language for creating web pages and web applications. [^1] #glossary 
- It is a subset of SGML and includes the mechanisms to establish hyperlinks to other documents. See also: hypertext, hyperlink, Standardized General Markup Language. [^2]
