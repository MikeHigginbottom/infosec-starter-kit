# Berkeley Software Distribution (BSD)
- Implementation of the UNIX operating system and its utilities developed and distributed by the University of California at Berkeley. BSD is usually preceded by the version number of the distribution, e.g., 4.3 BSD is version 4.3 of the Berkeley UNIX distribution. Many Internet hosts run BSD software, and it is the ancestor of many commercial UNIX implementations. [^1] #glossary
