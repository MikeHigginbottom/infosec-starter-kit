# Internet Message Control Protocol
- An Internet Standard protocol that is used to report error conditions during IP datagram processing and to exchange other information concerning the state of the IP network. [^1] #glossary 
- ICMP is an extension to the Internet Protocol. It allows for the generation of error messages, test packets and informational messages related to IP. [^2] #glossary
