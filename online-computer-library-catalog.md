# Online Computer Library Catalog
- OCLC is a nonprofit membership organization offering computer- based services to libraries, educational organizations, and their users. The OCLC library information network connects more than 10,000 libraries worldwide. Libraries use the OCLC System for cataloging, interlibrary loan, collection development, bibliographic verification, and reference searching. [^1] #glossary
