# MAC Address
- A media access control (MAC, IEEE 802) address is a usually (though not guaranteed) unique identifier assigned to a device's network interface controller. Also known as hardware or physical addresses. [^1] #glossary
- The hardware address of a device connected to a shared media. [^2] #glossary
