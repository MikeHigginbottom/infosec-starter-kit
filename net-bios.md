# NetBIOS
- Network Basic Input/Output System, or NetBIOS, is a session-layer application programming interface used to allow applications on separate devices to communicate over a local area network. NetBIOS uses ports 137 (TCP/UDP), 138 (UDP) and 139 (TCP). [^1] #glossary
