# fragment
- A piece of a packet. When a router is forwarding an IP packet to a network that has a maximum transmission unit smaller than the packet size, it is forced to break up that packet into multiple fragments. These fragments will be reassembled by the IP layer at the destination host. See also: Maximum Transmission Unit. [^1] #glossary
