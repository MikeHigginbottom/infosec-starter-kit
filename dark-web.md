# Dark Web
- Web content which is both not indexed by conventional search engines and can only be accessed using anonymity tools like the [TOR network](tor.md). [^1] #glossary

- The dark web should be contrasted with the [deep web](deep-web.md)
- The anonymity provided can be used for both good and bad causes, including protecting communications made by subjects of oppressive regimes or protecting the identity of criminals.   [^1]
