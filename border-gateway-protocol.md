# Border Gateway Protocol
- BGP is used to exchange routing information for the Internet and is the protocol used between [Internet service providers (ISP)](internet-service-provider.md) [^1] #glossary
- An inter-[autonomous system](autonomous-system.md) [exterior gateway routing protocol](exterior-gateway-protocol.md). [^1]
- Defined in RFC 1771. [^2]
- It's design is based on experience gained with EGP, as defined in RFC 904, and EGP usage in the NSFNET Backbone, as described in RFCs 1092 and 1093. [^2]
