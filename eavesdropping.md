# Eavesdropping
- An attack in which the attacker listens passively to supposedly secret transmissions, transactions, communications, data transfers or conversations in order to perpetrate an attack.  [^2] [^1] #glossary

- Eavesdropping can be used to refer to both data packet capture on a network link (also known as sniffing or packet capture) and to audio recording using a microphone (or listening with ears). [^2]
