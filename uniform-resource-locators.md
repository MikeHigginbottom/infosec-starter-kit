# Uniform Resource Locators (URL)
- A URL is a compact (most of the time) string representation for a resource available on the Internet. URLs are primarily used to retrieve information using WWW. The syntax and semantics for URLs are defined in RFC 1738. See also: World Wide Web. [^1] #glossary
