# DNS Reflection
- A Denial of Service (DoS) attack where an adversary sends a malicious Domain Name Service (DNS) request to a [DNS server](dns-server.md) that fools the [server](server.md) into responding instead to the victim of the attack. The origin of the attack is concealed from the victim. [^1] #glossary
