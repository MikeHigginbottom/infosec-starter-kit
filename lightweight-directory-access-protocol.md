# Lightweight Directory Access Protocol
- Defined in IETF RFC 4511, [LDAP](ldap.md) is an open-source application protocol used to access and maintain directory services across an IP network. By default, LDAP traffic uses TCP/UDP port 389. [^1] #glossary
