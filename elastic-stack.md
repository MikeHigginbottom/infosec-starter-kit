# Elastic Stack
- A [SIEM](siem.md) tool
- What's the difference between ELK Stack and Elastic Stack? ELK is the old term and refers to Elastic Search, Logstash and Kibana. Additional tools, e.g. 'beats', have been integrated into Elastic's releases so the whole package is now referred to as Elastic Stack. [^1]
- 'beats' are software that are used to move the data into elasticsearch. So you have winlogbeats to move windows logs into elasticsearch, amazonbeats which will read info about amazon products and copy the data into elasticsearch, etc.
