# Frame
- A frame is a datalink layer [packet](packet.md) which contains the header and trailer information required by the physical medium. That is, network layer packets are encapsulated to become frames. #glossary 
- See also: [datagram](datagram.md), [encapsulation](encapsulation.md), [packet](packet.md) [^2] #glossary
- Data that is transmitted between network points as a unit complete with addressing and necessary protocol control information. [^2]
- A frame is usually transmitted serial bit by bit and contains a header field and a trailer field that "frame" the data. [^1]
- Some control frames contain no data. [^1]
