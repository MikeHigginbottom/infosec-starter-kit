# Asynchronous Transfer Mode (ATM)
- A standard which defines high-load, high-speed (1.544Mbps through 1.2Gbps), fixed-size packet (cell) switching with dynamic bandwidth allocation. ATM is also known as fast packet. [^1] #glossary
