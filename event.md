# Event
- Any observable occurrence in a network or information system. [^2] An Event is the potential precursor to damage but this does not manifest. [^1] #glossary 

- See also [Event Severity Scale](event-severity-scale.md)
- An occurrence relating to security that's sufficiently interesting that you think it's worth recording for later reference or reporting. [^1]
