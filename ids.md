# IDS
- Program or device used to detect that an attacker is or has attempted unauthorised access to computer resources. [^2]
- An IDS is considered a more passive security tool as it detects compromises after they are already occurring rather than preventing them from becoming successful (this is the role of an [IPS](ips.md). [^1] #glossary
- There are two distinct types of IDS: [^3]
  - [Network-based intrusion detection system (NIDS)](network-based-ids.md) - NIDS are placed strategically within a network to monitor all traffic passing through and between devices on the network.
  - [Host-based intrusion detection system (HIDS)](host-based-ids.md) - HIDS are placed on individual devices within a network and monitor all inbound and outbound traffic to the device.
- See also [SIEM](siem.md)
