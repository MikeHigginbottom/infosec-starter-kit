# Autonomous System
- One network or series of networks that are all under one administrative control. [^1] #glossary
- Using a common Interior Gateway Protocol for routing packets. [^2]
- An autonomous system is also sometimes referred to as a routing domain. [^1]
- An autonomous system is assigned a globally unique number, sometimes called an Autonomous System Number (ASN). [^1]
