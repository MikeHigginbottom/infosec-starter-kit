# Dynamic Link Library
- A dynamic-link library is a Microsoft implementation of the shared-file format that contains code and date used by more than one program. [^1] A collection of small programs, any of which can be called when needed by a larger program that is running in the computer. The small program that lets the larger program communicate with a specific device such as a printer or scanner is often packaged as a DLL program (usually referred to as a DLL file). [^2] #glossary
