# Knowbot
- A Knowledge Robot is a program which seeks out information based on specified criteria. Knowbot, as trademarked by CNRI, refers specifically to the search engine for Knowbot Information Services. See also: Corporation for National Research Initiatives, X.500, white pages, whois, netfind. [^1] #glossary
