# Interior Gateway Protocol (IGP)
- A protocol which distributes routing information to the routers within an autonomous system. The term gateway is historical, as router is currently the preferred term. See also: Autonomous System, Exterior Gateway Protocol, Open Shortest-Path First, Routing Information Protocol. [^1] #glossary
