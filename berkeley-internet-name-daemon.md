# Berkeley Internet Name Daemon (BIND)
- Implementation of a DNS server developed and distributed by the University of California at Berkeley. Many Internet hosts run BIND, and it is the ancestor of many commercial BIND implementations. See also: Domain Name System. [^1] #glossary
