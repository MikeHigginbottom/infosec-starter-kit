# Point To Point Protocol
- A protocol for communication between two computers using a serial interface, typically a personal computer connected by phone line to a server. It packages your computer's TCP/IP packets and forwards them to the server where they can actually be put on the Internet. [^1] #glossary 
- The Point-to-Point Protocol, defined in RFC 1661, provides a method for transmitting packets over serial point-to-point links. There are many other RFCs which define extensions to the basic protocol. See also: Serial Line IP. [^2] #glossary
