# WPAD
- The Web Proxy Auto-Discovery Protocol (WPAD) is a method used by [clients](client.md) to locate the URL of a configuration file using DHCP and/or DNS discovery methods. Once detection and download of the configuration file is complete, it can be executed to determine the proxy for a specified URL. [^1] #glossary
