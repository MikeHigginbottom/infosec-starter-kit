# TCP/IP Protocol Suite
- Transmission Control Protocol over Internet Protocol. This is a common shorthand which refers to the suite of transport and application protocols which runs over IP. See also: IP, ICMP, TCP, UDP, FTP, Telnet, SMTP, SNMP. [^1] #glossary
