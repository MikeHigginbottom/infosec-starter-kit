# Drive-by Download
- A type of web-based attack that automatically occurs based on the simple act of visiting a malicious or compromised/poisoned Web site. [^2] A download which a user is not aware of or has not consented to. Commonly used to refer to malware downloaded from compromised legitimate websites. [^1] #glossary

- Accomplished by taking advantage of the default nature of a Web browser to execute mobile code, most often JavaScript, with little to no security restrictions. [^2] 
- Can install tracking tools, remote access backdoors, botnet agents, keystroke loggers or other forms of malicious utilities. [^2]
