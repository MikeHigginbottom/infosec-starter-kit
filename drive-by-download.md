# Drive-by Download
- A type of web-based attack that automatically occurs based on the simple act of visiting a malicious or compromised/poisoned Web site. #ref/2023/12/09/130055 A download which a user is not aware of or has not consented to. Commonly used to refer to malware downloaded from compromised legitimate websites. #ref/2023/12/09/120659 #glossary

- Accomplished by taking advantage of the default nature of a Web browser to execute mobile code, most often JavaScript, with little to no security restrictions. #ref/2023/12/09/130055 
- Can install tracking tools, remote access backdoors, botnet agents, keystroke loggers or other forms of malicious utilities. #ref/2023/12/09/130055
