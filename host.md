# Host
- Any computer that has full two-way access to other computers on the Internet. Or a computer with a [web server](web-server.md) that serves the pages for one or more Web sites. [^1] #glossary 
- A computer that allows users to communicate with other host computers on a network. Individual users communicate by using application programs, such as electronic mail, Telnet and FTP. [^2]
