# routed
- Route Daemon. A program which runs under 4.2BSD/4.3BSD UNIX systems (and derived operating systems) to propagate routes among machines on a local area network, using the RIP protocol. Pronounced route-dee. See also: Routing Information Protocol, gated. [^1] #glossary
