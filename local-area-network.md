# Local Area Network (LAN)
- A data network intended to serve an area of only a few square kilometers or less. Because the network is known to cover only a small area, optimizations can be made in the network signal protocols that permit data rates up to 100Mb/s. See also: Ethernet, Fiber Distributed Data Interface, token ring, Metropolitan Area Network, Wide Area Network. [^1] #glossary
