# X Forwarded For Header
- The X-Forwarded-For ([XFF](xff.md)) HTTP Header field is used to identify the original IP address of a [client](client.md) machine connecting to a [server](server.md) through a [proxy server](proxy-server.md) or load balancer. [^1] #glossary
