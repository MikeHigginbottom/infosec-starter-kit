# Simple Mail Transfer Protocol (SMTP)
- A protocol used to transfer electronic mail between computers. It is specified in RFC 821, with extensions specified in many other RFCs. It is a server to server protocol, so other protocols are used to access the messages. See also: Electronic Mail, Post Office Protocol, RFC 822. [^1] #glossary
