# Asymmetric Cryptography
- A mechanism whereby each party in a data interchange advertises a [public key](public-key.md) for the other to use for encryption, whilst using a [private key](private-key.md) to decrypt whatever arrives. [^1] #glossary

- Largely synonymous with [Public Key Cryptography](public-key-cryptography.md) [^1]
- [Rivest-Shamir-Adleman (RSA)](rivest-shamir-adelman.md)
