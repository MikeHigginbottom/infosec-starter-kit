# Client
- A system entity (computer system or process) that requests and uses a service provided by another system entity, called a [server](server.md) In some cases, the server may itself be a client of some other server. [^1] #glossary
- A workstation requesting the contents of a file from a file server is a client of the file server. [^2]
- See also: [client-server model](client-server-model.md), [server](server.md)
