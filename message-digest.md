# Message Digest
- The output after applying a [hash function](hashing.md) to some [clear text](clear-text.md). [^1] #glossary 
- Message digests are algorithmic operations, generally performed on text, which produce a unique signature for that text. MD-2, described in RFC 1319; MD-4, described in RFC 1320; and MD-5, described in RFC 1321 all produce a 128-bit signature. They differ in their operating speed and resistance to crypto-analytic attack. Generally, one must be traded off for the other. [^2] #glossary
