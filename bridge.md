# Bridge
- A product that connects a local area network (LAN) to another local area network that uses the same protocol (for example, Ethernet or token ring). [^1] #glossary 
- Forwards traffic between network segments based on datalink layer information. These segments would have a common network layer address. See also [gateway](gateway.md), [router](router.md). [^2]
