# Admin User
- A high-privilege login account that is able to do more than a normal user. System administrators use them to reconfigure systems, create and delete normal user accounts, and so on. [^1] #glossary

- The System Administrator is a technical specialist who manages a computer system, and whose credentials are usually highly privileged and hence sought after by Hackers. [^1]
- `root` is the name of the administrator account in Unix systems. [^2]
