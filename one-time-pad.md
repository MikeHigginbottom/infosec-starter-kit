# One Time Pad
- A paper pad of encryption Keys on which each key is different from the next and where there is no discernible pattern to the various keys. The sender and receiver must have the same One-Time Pads in order that the receiver can decrypt the message. One of the most secure approaches to an Encryption Algorithm so long as nobody is able to duplicate the One Time Pad. [^1] #glossary
