# Open Shortest Path First
- Open Shortest Path First is a link state routing algorithm used in interior gateway routing. Routers maintain a database of all routers in the autonomous system with links between the routers, link costs, and link states (up and down). [^1] #glossary 
- A link state, as opposed to distance vector, routing protocol. It is an Internet standard IGP defined in RFCs 1583 and 1793. The multicast version, MOSPF, is defined in RFC 1584. See also: Interior Gateway Protocol, Routing Information Protocol. [^2] #glossary
