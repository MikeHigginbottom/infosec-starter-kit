# Post Office Protocol
- An Internet Standard protocol by which a client workstation can dynamically access a mailbox on a server host to retrieve mail messages that the server has received and is holding for the client. [^1] #glossary 
- A protocol designed to allow single user hosts to read electronic mail from a server. Version 3, the most recent and most widely used, is defined in RFC 1725. See also: Electronic Mail. [^2] #glossary
