# Management Information Base (MIB)
- The set of parameters an SNMP management station can query or set in the SNMP agent of a network device (e.g. router). Standard, minimal MIBs have been defined, and vendors often have Private enterprise MIBs. In theory, any SNMP manager can talk to any SNMP agent with a properly defined MIB. See also: client-server model, Simple Network Management Protocol. [^1] #glossary
